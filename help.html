1. Animegt will adapt the source that you selected and will find every anime & servers of it.

2. You can change sources trough the world icon on the main page.

3. If you change the source the animes on your favorites will not change the source from where you add them, the little letters above the title indicates the source.

4. Each source has its own animes & servers, If you dont find an anime you can change the source and search it again, if you still dont find it try using it's japanese name.

5. If the app doesn't search or show anything that means that your current source it`s not available and you should change it following the instructions above.
